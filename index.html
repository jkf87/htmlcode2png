사용법

  1. 왼쪽 텍스트 영역에 HTML 코드를 붙여넣습니다
  2. "테스트 HTML 로드" 버튼으로 샘플 코드를 확인할 수 있습니다
  3. "PNG로 변환" 버튼을 클릭합니다
  4. 미리보기에서 결과를 확인하고 "PNG 다운로드" 버튼을 클릭합니다
